gitment配置过程中出现的问题

发表于 2019-02-27 更新于 2019-02-28

在hexo博客配置gitment评论系统时，出现Object ProgressEvent或validation failed的解决方法。

Object ProgressEvent

打开blog目录中themes\yilia\layout\_partial\post\gitment.ejs将

修改为

更新hexo clean&&hexo g,同步hexo d,再重新打开网站登录就不会报错了，而且评论框也已经汉化了。

validation failed

打开blog目录中themes\yilia\layout\_partial\post\gitment.ejs将
